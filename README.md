# Tax Calculation System

An automated income tax calculation system designed to help users calculate their income tax based on **Indian tax laws**. It provides both salaried and self-employed individuals with a simple, accurate, and user-friendly tax computation experience.

## ✨ Key Features

- **Dual Tax Regime Support** — Calculates tax under both Old and New Indian tax regimes
- **Multiple Income Types** — Supports salaried, self-employed, and other income categories
- **Deduction Calculator** — Accounts for Section 80C, 80D, HRA, and other common deductions
- **Real-Time Calculation** — Instant tax computation as users input their financial data
- **Clean UI** — Simple, intuitive interface for hassle-free tax planning

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | HTML5, CSS3, JavaScript |
| Logic | Vanilla JavaScript (client-side computation) |
